# 🚀 Projetos Front-End - DevMedia

Este repositório reúne uma série de projetos práticos desenvolvidos por mim durante os cursos de formação Front-End da [DevMedia](https://www.devmedia.com.br/). O objetivo principal desta série foi solidificar a base em **HTML5** e **CSS3**, aplicando conceitos que vão desde a estruturação semântica até a estilização avançada e responsividade.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de páginas.
* **CSS3:** Estilização, layout (Flexbox/Grid), transições e responsividade.
* **Mobile-First / Responsividade:** Práticas para adaptação de telas (Media Queries).
